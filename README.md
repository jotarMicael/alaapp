# creativescienceapp

Se deben instalar las siguientes dependencias:
  
  python-dotenv 
  python-decouple
  
Se debe crear un archivo .env con las siguientes variables para confirgurar el entorno:

  #DB
  MYSQL_HOST='valor'
  MYSQL_USER='valor'
  MYSQL_PASSWORD='valor'
  MYSQL_DB='valor'
  MYSQL_PORT=''

  #SERVICE_EMAIL
  EMAIL_BACKEND='valor'
  EMAIL_HOST='valor'
  EMAIL_USE_TLS='valor'
  EMAIL_PORT='valor'
  EMAIL_HOST_USER='valor'
  EMAIL_HOST_PASSWORD='valor'
